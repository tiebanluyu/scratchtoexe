Scratch3.0转EXE

Scratch3.0转EXE很复杂，但目前有两种方法

1. 技术羊方案，点击看教程

其实它就是对方案二的自动化方案

 

2.经典方案

下载工具1：makesfx.exe。无需安装，双击即可运行。

下载地址：甲地址http://revocue.cz/en/make-sfx/index.php

丙https://share.weiyun.com/E3VtktDM

 

 

下载工具2：nwjs。下载地址：https://nwjs.io/downloads/

源地址慢死，网盘链接：https://share.weiyun.com/NYoI3Sn8

开始

打开https://sheeptester.github.io/htmlifier/       这是原作者的

站主备份地址https://tiebanluyu.github.io/scratchtoexe/sb3tohtml.html

对，都放在GitHub page上

根据提示，将sb3文件转换成html文件（英文警告）

将nwjs的压缩包解压成文件夹，将刚才的html文件丢进去

在文件夹内创建package.json

内容如下

{

"name": "您的应用程序或游戏名称",

"main": "result.html"

}

　

（默认你不用图标）

要的话

{

"name": "您的应用程序或游戏名称",

"main": "result.html",

"icons": {"16":"01.ico"}

}

　


　

   

01.ico是图标文件

result.html是转换成的文件

可以换

双击运行nw.exe

其实现在把文件夹丢给别人，让他点nw.exe

也行，但出于方便

运行makesfx.exe，指定目录、文件名和图标，制作成一个exe文件

配张图

这样就大功告成了
